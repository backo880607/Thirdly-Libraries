IPdfFontFile Class

Implements
IPdfFontAttributes
Module
wijmo.pdf
File
wijmo.pdf.js

Represents the settings of the font to register by registerFont and registerFontAsync methods.

Properties

Name Type Description
F family? string An optional parameter determining the TrueType Collection or Datafork TrueType font family.
F name string The name of the font to use.
F source any An ArrayBuffer containing binary data or URL to load the font from. Following font formats are supported: TrueType (.ttf), TrueType Collection (.ttc), Datafork TrueType (.dfont).
F style? string The style of the font. One of the following values: 'normal', 'italic', 'oblique'.
F weight? string The weight of the font. One of the following values: 'normal', 'bold', '100', '200', '300', '400', '500', '600', '700', '800', '900'.
