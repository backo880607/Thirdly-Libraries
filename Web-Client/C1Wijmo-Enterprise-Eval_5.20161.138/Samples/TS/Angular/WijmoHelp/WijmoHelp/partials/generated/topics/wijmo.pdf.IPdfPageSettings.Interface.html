IPdfPageSettings Class

Module
wijmo.pdf
File

Represents the page settings.

Properties

Name Type Description
F layout? PdfPageOrientation Determines the layout of the page.
F margins? IPdfPageMargins Determines the margins of the page.
F size? any Determines the dimensions of the page. The following values are supported:
  • PdfPageSize: predefined sizes.
  • Size: custom sizes.
