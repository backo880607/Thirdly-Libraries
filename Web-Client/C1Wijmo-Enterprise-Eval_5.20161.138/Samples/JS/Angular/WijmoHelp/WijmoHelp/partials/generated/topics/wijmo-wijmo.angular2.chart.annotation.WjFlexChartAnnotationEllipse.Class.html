Angular 2 component for the WjFlexChart Ellipse annotation objects.

The wj-flex-chart-annotation-ellipse component must be contained in a WjFlexChartAnnotationLayer component.

Use the wj-flex-chart-annotation-ellipse component to add Ellipse annotation objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartAnnotationEllipse component is derived from the Ellipse class and inherits all its properties, events and methods.
