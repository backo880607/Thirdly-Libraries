Represents the animation for FlexChart, FinancialChart and FlexPie.

The ChartAnimation provides built-in animation while loading and updating the chart. The animation can be configured by the user through several properties that include duration, easing function, animation mode.

Properties

Name Type Description
animationMode AnimationMode Gets or sets whether the plot points animate one at a time, series by series, or all at once. The whole animation is still completed within the duration.

duration number Gets or sets the length of entire animation in milliseconds.
easing Easing Gets or sets the easing function applied to the animation.
